# 📖 PDF Chatbot with Streamlit & FLAN-T5

A beginner-friendly chatbot that lets you upload a PDF and ask questions about its content. It uses [LangChain](https://www.langchain.com/), [Hugging Face Transformers](https://huggingface.co/), and [Streamlit](https://streamlit.io/).

---

## 🚀 Features

- Upload any PDF
- Automatically extracts and splits text
- Generates embeddings using Sentence Transformers
- Uses FLAN-T5 model to answer questions based on context
